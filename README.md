# Proyecto de ayuda emocional

Aplicación web personal creada como regalo. El proyecto reúne distintos tipos de contenido personal en una experiencia sencilla con recuerdos, mensajes, música, razones de apoyo y recursos para momentos difíciles.

Fue desarrollado con HTML, CSS y JavaScript, sin frameworks ni backend.

## Demo

Puedes ver el proyecto publicado aquí:

## Funcionalidades

- Página de inicio con mensaje principal.
- Galería de fotografías organizada por álbumes.
- Reproductor o playlist musical.
- Mensajes según el estado emocional.
- Lista aleatoria de razones y mensajes positivos.
- Sección S.O.S. con imágenes, frases, audios y contenido de distracción.
- Navegación entre las diferentes secciones.
- Diseño adaptado para móvil y ordenador.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub Pages
